## Updating the progress tracker

`docs/ReversedClasses.md` is auto-generated. To regenerate it after reversing functions, run the generation script from the project root. Do not edit the file by hand.

When you reverse a function, set `reversed: true` on its `RH_ScopedInstall` call (or omit the options struct entirely, since `true` is the default). The progress tracker reads this metadata to compute completion percentages.

```cpp
// Stub — not counted as reversed
RH_ScopedInstall(FireInstantHit, 0x73FB10, { .reversed = false });

// Complete — counted as reversed
RH_ScopedInstall(FireInstantHit, 0x73FB10);
```
