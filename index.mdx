## Welcome to GTA:SA Reversed

GTA:SA Reversed is an open-source project that reimplements Grand Theft Auto: San Andreas (1.0 US) function-by-function in modern C++23. The result is a DLL that injects into the game via an ASI loader, replacing original game functions with fully documented, reversible equivalents — while keeping the game running identically to the original.

With **~7,965 functions** reversed across **700 classes** (approximately 79% complete), this is one of the most comprehensive reverse-engineering efforts in the GTA modding community.

## How it works

<Steps>
  <Step title="Build the DLL">
    Clone the repository, install dependencies with Conan, generate a Visual Studio solution with CMake, and compile `gta_reversed.asi`.
  </Step>
  <Step title="Install the ASI loader">
    Place Silent's ASI Loader into your GTA:SA directory so it automatically loads `.asi` files on game start.
  </Step>
  <Step title="Run the game">
    Launch GTA:SA with the Compact exe. The DLL injects and replaces original functions with reversed implementations — the game runs as normal.
  </Step>
  <Step title="Modify and test">
    Edit reversed C++ code, recompile, and attach a debugger via ReAttach. Changes take effect immediately in the running game.
  </Step>
</Steps>
